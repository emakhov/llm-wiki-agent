# Knowledge Base Overview

This is a personal knowledge base maintained by an AI agent.

## How It Works

1. **Add sources** — drop articles, papers, or notes into `sources/` within this vault (e.g. via Obsidian Web Clipper).
2. **Ingest** — tell the agent to process a source. It reads the document, creates summary and entity pages, updates cross-references, and logs the operation.
3. **Query** — ask questions. The agent searches the wiki, reads relevant pages, and synthesizes answers with citations.
4. **Lint** — ask the agent to health-check the wiki. It finds orphan pages, broken links, and missing cross-references.
